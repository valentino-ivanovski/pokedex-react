# pokedex-react
The entire Pokédex built in React with the help of pokkeapi.co

1. Features a list of all the Kanto Pokemon on the sidebar, with a search bar.
2. When a pokemon is clicked their info is shown, alongside with a picture, their moves, and other things which are retrieved from the pokeapi.co API.
3. The app provides both a light and dark mode theme.
